### Context
개인 프로젝트나 일상에서 발생하는 주식, 쇼핑몰 운영, 코딩 기술 등 파편화된 정보를 체계적인 지식 자산으로 전환하기 위해 n8n과 AI 에이전트를 결합한 '개인 인사이트 관리 시스템'을 설계함. 단순 기록을 넘어 AI가 내용을 분석하고 카테고리(스토어, 주식, 쇼핑몰, n8n, 코딩, 판매DB 등)를 자동 분류하여 데이터베이스화하는 것이 목표임.

### Core
n8n의 AI Agent 노드와 툴 호출(Tool Calling) 기능을 활용하여 입력된 메모를 분석하고 구조화된 데이터로 변환하는 워크플로우의 핵심 로직임.

*   **입력 단계**: Webhook 또는 Telegram 노드를 통해 단편화된 텍스트 수신
*   **분석 단계 (AI Agent Node)**: LLM(예: GPT-4o 또는 Claude 3.5 Sonnet)을 사용하여 다음 프롬프트를 실행
    ```text
    사용자의 입력 문장을 분석하여 [주식, 쇼핑몰, n8n, 코딩, DB] 중 하나로 카테고리를 분류하세요.
    내용에서 핵심 인사이트를 추출하고, 기술적 용어는 표준화하여 JSON 형식으로 반환하세요.
    ```
*   **저장 단계**: 분류된 데이터를 기반으로 조건부 분기(If Node) 또는 스위치(Switch Node)를 사용하여 목적지 DB(PostgreSQL 또는 Google Sheets)에 적재

### Insight
다양한 도메인의 정보를 하나의 시스템에서 관리할 때 가장 중요한 것은 '분류의 정확성'과 '데이터의 재사용성'임. 단순 텍스트 저장이 아닌 AI를 통한 구조화 과정을 거치면, 추후 특정 섹션(예: 판매DB)의 데이터만 추출하여 통계 분석을 하거나 추가적인 자동화 워크플로우의 트리거로 활용하기 용이함. 특히 n8n의 AI Agent 노드를 활용하면 복잡한 코딩 없이도 자연어 기반의 정교한 데이터 분류 체계를 구축할 수 있다는 이점이 있음.
